# Gallerist Car Sales Platform 🚗

A backend system for managing car sales, built with **Spring Boot**, **PostgreSQL**, and **JWT-based security**.  

This project handles car listings, sales operations, and secure API endpoints for seamless management.

---

## 🛠 Technologies Used
- Spring Boot  
- Spring Data JPA  
- PostgreSQL  
- REST API  
- DTO & Validation  
- JWT Authentication & Spring Security  
- JUnit 5  

---

## ✨ Features
- CRUD operations for car listings and sales  
- REST API endpoints for secure data access  
- Data integrity with validation and entity relationships (`@OneToOne`, `@OneToMany`, `@ManyToOne`)  
- JWT authentication and Spring Security for API protection  
- Unit testing using JUnit 5
